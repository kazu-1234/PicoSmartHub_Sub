# PicoSmartHub_Sub

PicoSmartHub_Subは、Raspberry Pi Pico W 2（およびPico W）を使用した、PicoSmartHubシステムの専用子機デバイスです。
超音波センサーを用いた水位監視をメイン機能とし、ネットワークを介して親機（PicoSmartHub）と連携することで、家の中の離れた場所へ状態を通知します。

### 電源供給について
本機は設置の柔軟性を高めるため、「USB給電」と「乾電池」の2種類の電源方式に対応しています。
**USBケーブルによる給電と、乾電池による給電を絶対に同時に行わないでください。**

## 主な機能

### 1. お風呂の湯張り監視（メイン機能）
超音波距離センサー（HC-SR04等）を利用して、浴槽内の水位をリアルタイムで計測します。
- **自動検知と通知**: 水面までの距離が設定値（しきい値）以下になったことを検知すると、即座にネットワーク経由で親機へ通知を送ります。
- **親機との連携**: 独自の通信プロトコルにより、親機（[PicoSmartHub](https://github.com/kazu-1234/PicoSmartHub)）の画面表示を上書きし、LED点滅とともに「お風呂が沸いたこと」を家族に知らせます。これにより、浴室から離れたリビングや自室にいても湯張りの完了を確実に把握できます。

### 2. 環境データの計測と送信
子機周辺の温度・湿度を測定し、親機へデータを共有したり、親機を経由してクラウド（Google Apps Script）へログを送信したりすることが可能です。親機側で家全体の環境を一括管理するための一部門として機能します。

### 3. ステータス表示
小型のLCDやLED（構成による）を搭載し、現在のWi-Fi接続状況や水位計測の状態を現場でも確認できるように設計されています。

## システム構成・ピンアサイン

Raspberry Pi Pico W 2 (Pico W 互換) における標準的なピン接続です。

- **超音波距離センサー (HC-SR04)**
  - Trigger: GP14
  - Echo: GP15
- **温湿度センサー (DHT11/22)**
  - Data: GP16
- **ステータスLED**
  - GP17 (または内蔵LED)
- **I2C通信 (ディスプレイ等)**
  - SDA: GP0
  - SCL: GP1

## 導入・セットアップ

1. **開発環境**
   Arduino IDEを使用し、ボードマネージャから Raspberry Pi Pico / RP2040 (earlephilhower版) を選択してください。
2. **ライブラリ**
   以下のライブラリを事前にインストールしてください。
   - `DHT sensor library`
   - `ArduinoJson`
3. **設定（config.h）**
   `config.h` 内の以下の項目を環境に合わせて書き換えてください。
   - 接続先Wi-FiのSSID/パスワード
   - 親機（PicoSmartHub）の固定IPアドレス
   - 水位検知のしきい値（距離cm）
4. **書き込み**
   設定完了後、USB経由でプログラムを書き込みます。
